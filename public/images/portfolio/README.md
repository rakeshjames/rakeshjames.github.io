# Drop your portfolio images in this folder

Supported formats: .jpg, .jpeg, .png, .webp, .gif

After adding images:
1. Open `data/portfolio.js`
2. Add an entry to `portfolioItems` pointing to your new file
   e.g.  image: '/images/portfolio/my-new-image.jpg'
3. Commit and push → GitHub Actions will redeploy automatically
